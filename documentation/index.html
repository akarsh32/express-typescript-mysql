expressts

ExpressJS with MySQL in Typescript

Installation

Simply run

$ npm install

or via Yarn.

$ yarn

Running in development

 
$ npm run dev

Type rs inside the terminal to manually restart the app.

more information ts-node-dev

Production

 
$ npm run build

Tests

 
$ npm run test

This will build all files inside dist folder
